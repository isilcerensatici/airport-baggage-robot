# Airport Baggage Robot Project

This project was assigned by Dr. Lecturer Hasan Serdar for the Robotics course in the 4th grade, 2nd semester.

In this project, an autonomous airport baggage robot simulation was developed using ROS and Gazebo. The robot was designed based on TurtleBot3 and tested in a custom airport-like environment created in Gazebo.

During this project, I learned:

- ROS (Robot Operating System) fundamentals
- Gazebo robot simulation environment
- TurtleBot3 robot structure
- SLAM (Simultaneous Localization and Mapping)
- Gmapping SLAM algorithm
- Autonomous exploration systems
- Basic robot navigation and obstacle avoidance
- ROS launch files and package structure

Technologies and tools used in this project:

- ROS Noetic
- Gazebo
- TurtleBot3
- Gmapping SLAM
- explore_lite
- Python
- Ubuntu Linux

The robot autonomously explored the environment and generated a map of the airport area successfully.
